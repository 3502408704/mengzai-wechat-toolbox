## v0.8 公测版 (2026-05-24)

### 🎨 全新 UI（移植自排版助手原版）

- **全新配色体系** — 蓝色/紫蓝/琥珀三色品牌色，支持明暗主题
- **编辑器重构** — 完整 Markdown 格式工具栏（13 个格式按钮：加粗、斜体、删除线、标题、引用、无序列表、有序列表、链接、图片、行内代码、代码块、分割线、撤销/重做）
- **模板选择** — 极简风（经典/清晨/暮光）+ 商务风（经典/深蓝/沉稳）+ 文艺风（春风/暖阳/夜曲）+ 科技风（终端/脉冲/回响）
- **历史页面** — 卡片式列表，支持预览/编辑/删除/清空
- **底部导航精简** — 编辑 · 历史 · 设置 三个标签
- **使用指南** — 完整帮助页面，含快速入门、AI 辅助、历史记录、无障碍使用说明和常见问题

### 🤖 AI 辅助

- 聊天气泡式消息，左对齐 AI / 右对齐用户，颜色区分
- 代码块独立复制
- 消息流式播报，支持 LiveRegion 自动播报
- 多模型配置支持（设置页添加 AI 配置）

### ♿ 无障碍增强

- 所有可聚焦控件添加语义标签（contentDescription）
- 格式按钮选中/未选中状态可被屏幕阅读器感知
- Snackbar 消息通过 LiveRegion 自动朗读
- 触觉反馈支持
- 符合 WCAG AA 标准
